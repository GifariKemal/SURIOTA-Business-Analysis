## Product Overview

**SURGE** is a multi-tenant IoT SaaS platform for industrial monitoring with 3 main modules:

| Module | Use Case | Key Features |
|--------|----------|--------------|
| **Water Analytics** | IPAL, STP, WTP, PDAM | KLHK compliance, real-time monitoring |
| **Energy Mapping** | Buildings, Factories | kWh monitoring, cost allocation |
| **Vessel Tracking** | Fleet, Maritime | GPS tracking, fuel monitoring |

### Pricing Tiers

| Plan | Price/Month | Parameters | Locations | Retention |
|------|-------------|:----------:|:---------:|:---------:|
| Trial | Free | 5 | 1 | 30 days |
| Starter | $29 (Rp 464K) | 10 | 3 | 90 days |

### Key Differentiators

1. **KLHK Compliance** - Built-in export format for SISPEK
2. **Bahasa Indonesia** - UI and support in Indonesian
3. **Local Support** - WhatsApp and phone support
4. **3 Modules in 1** - Water, Energy, Vessel integrated
5. **Affordable** - 75% cheaper than ThingsBoard
