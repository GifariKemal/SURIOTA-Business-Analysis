## 1. Naming Criteria

### 1.1 Naming Requirements

| Criteria         | Importance | Description                            |
| ---------------- | ---------- | -------------------------------------- |
| **Meaningful**   | High       | Reflects care/health/companion concept |
| **Memorable**    | High       | Easy to remember and pronounce         |
| **Short**        | High       | Max 3 syllables, easy to type          |
| **Unique**       | High       | Not too similar to competitors         |
| **Available**    | High       | Domain and app store availability      |
| **Scalable**     | Medium     | Works beyond Batam market              |
| **Local Flavor** | Medium     | Resonates with Indonesian audience     |
| **SURIOTA Link** | Low        | Optional connection to parent company  |

### 1.2 Competitor Naming Patterns

| App           | Pattern                | Notes                      |
| ------------- | ---------------------- | -------------------------- |
| **Medi-Call** | Service + Action       | Medical + Call (summon)    |
| **Kavacare**  | Invented + Purpose     | Kava (?) + Care            |
| **CarePro**   | Purpose + Professional | Care + Professional        |
| **LoveCare**  | Emotion + Purpose      | Love + Care                |
| **Homage**    | Single meaningful word | Respect/tribute to elderly |
