### 5.1 Positioning Map

```
           Low        │

                      └─────────────────────────► COVERAGE
                    Limited              Wide
```

### 5.2 SRICARE Positioning Statement

> **For** perantau dan keluarga di Batam **who** membutuhkan pendamping terpercaya untuk pasien, **SRICARE** adalah aplikasi caregiving on-demand **that** menghubungkan dengan caregiver terverifikasi melalui GPS real-time. **Unlike** kompetitor Jakarta-based yang tidak cover Batam, **SRICARE** hadir dengan harga terjangkau, layanan pendampingan RS & cuci darah, dan support lokal berbahasa Indonesia.

### 5.3 Key Differentiators

| Differentiator         | SRICARE                  | Competitors     |
| ---------------------- | ------------------------ | --------------- |
| **Batam Focus**        | First & Only             | None present    |
| **Hospital Companion** | Core service             | Not offered     |
| **Dialysis Support**   | Core service             | Not offered     |
| **Real-time GPS**      | Yes (Gojek-style)        | Limited         |
| **Affordable Pricing** | 15-20% lower             | Premium pricing |
| **Local Support**      | WIB timezone, Batam team | Jakarta-centric |

## 6. Battlecard

### 6.1 Quick Reference - How to Win Against Each Competitor

#### vs. Medi-Call

| Their Strength      | Our Counter                      |
| ------------------- | -------------------------------- |
| Established brand   | Local presence + community trust |
| Multi-city coverage | Deep Batam specialization        |
| Medical focus       | Companion-first (their gap)      |

**Win Message:**

> "Medi-Call bagus untuk layanan medis di Jakarta, tapi untuk pendampingan pasien di Batam, SRICARE lebih dekat dan lebih paham kebutuhan lokal."

#### vs. Kavacare

| Their Strength      | Our Counter                          |
| ------------------- | ------------------------------------ |
| Geriatric specialty | Same capability + hospital companion |
| Medical tourism     | Not our focus (different segment)    |
| Premium packages    | Affordable per-service pricing       |

**Win Message:**

> "Kavacare fokus ke pasien premium Jabodetabek. Untuk keluarga di Batam yang butuh pendamping terjangkau, SRICARE adalah pilihan tepat."

#### vs. CarePro

| Their Strength    | Our Counter                  |
| ----------------- | ---------------------------- |
| Marketplace model | Same model, different market |
| DTO recognition   | SURIOTA tech credibility     |
| Verified workers  | Same verification standards  |

**Win Message:**

> "CarePro model marketplace yang bagus, tapi fokus Jabodetabek. Kami bawa model yang sama ke Batam - pasar yang belum mereka sentuh."

#### vs. LoveCare

| Their Strength    | Our Counter                       |
| ----------------- | --------------------------------- |
| 50+ city coverage | Deeper Batam network              |
| General care      | Specialized hospital + HD support |
| Quick matching    | Same + real-time tracking         |

**Win Message:**

> "LoveCare coverage luas tapi tipis. Di Batam, SRICARE punya jaringan caregiver lebih kuat dan layanan lebih spesifik."

#### vs. Informal Services

| Their Strength      | Our Counter                         |
| ------------------- | ----------------------------------- |
| Cheaper price       | Quality guarantee + insurance       |
| Personal connection | Verified caregivers + ratings       |
| Flexible            | On-demand booking, same flexibility |

**Win Message:**

> "Jasa informal mungkin lebih murah, tapi tidak ada jaminan. SRICARE: caregiver terverifikasi, ada tracking, bisa komplain kalau tidak puas."

## 7. Competitive Strategy

### 7.1 Short-term (6 months)

| Strategy               | Action                                                       |
| ---------------------- | ------------------------------------------------------------ |
| **First-mover**        | Establish strong presence in Batam before competitors expand |
| **Local partnerships** | MoU with RS and dialysis clinics                             |
| **Community building** | Build caregiver community, referral network                  |
| **Quality focus**      | High ratings = word of mouth                                 |

### 7.2 Medium-term (6-18 months)

| Strategy              | Action                                       |
| --------------------- | -------------------------------------------- |
| **Geographic moat**   | Expand to Kepri islands, Tanjung Pinang      |
| **Service expansion** | Add doctor visit, lab services               |
| **B2B channel**       | Corporate employee benefits program          |
| **Technology moat**   | Integration with SURGE for health monitoring |

### 7.3 Long-term (18+ months)

| Strategy                | Action                                |
| ----------------------- | ------------------------------------- |
| **Regional expansion**  | Sumatera (Medan, Pekanbaru)           |
| **Platform ecosystem**  | Integrate with health insurance, BPJS |
| **Data moat**           | Healthcare analytics, predictive care |
| **Acquisition defense** | Strong local brand, loyal users       |

### 7.4 Competitive Response Plan

**If Jakarta competitors enter Batam:**

1. **Accelerate user acquisition** - Heavy discounts, referral bonuses
2. **Lock in caregivers** - Exclusive partnership, higher commission
3. **Deepen hospital partnerships** - In-hospital promotion rights
4. **Emphasize local** - "Made in Batam, for Batam" messaging
5. **Niche specialization** - Double down on HD + hospital companion
