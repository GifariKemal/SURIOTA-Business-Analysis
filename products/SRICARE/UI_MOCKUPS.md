## 6.2 Accessibility

```
┌─────────────────────────────────────────────────────────────────┐
│                    ACCESSIBILITY                                │
├─────────────────────────────────────────────────────────────────┤
│                                                                 │
│  TOUCH TARGETS                                                  │
│  • Minimum size: 48x48dp                                        │
│  • Spacing between targets: 8dp minimum                         │
│                                                                 │
│  CONTRAST                                                       │
│  • Text on light background: 4.5:1 minimum                      │
│  • Large text (18sp+): 3:1 minimum                              │
│  • Icons: 3:1 minimum                                           │
│                                                                 │
│  TEXT SCALING                                                   │
│  • Support system font scaling up to 200%                       │
│  • Test layouts at all scale levels                             │
│                                                                 │
│  SCREEN READERS                                                 │
│  • All interactive elements have labels                         │
│  • Images have alt text                                         │
│  • Focus order is logical                                       │
│                                                                 │
│  COLOR BLINDNESS                                                │
│  • Don't rely on color alone for information                    │
│  • Use icons + text + color for status                          │
│                                                                 │
└─────────────────────────────────────────────────────────────────┘
```
