## Key Features (USP)

1. **BLE Mobile Configuration** - Setup via smartphone dalam 5 menit (EXCLUSIVE)
2. **WiFi Auto Failover** - Automatic switch Ethernet ke WiFi
3. **RTC with Battery Backup** - Timestamp akurat meski power loss
4. **9 LED Status Indicators** - Visual system monitoring lengkap
5. **ESD Protection** - Industrial grade RS-485 protection
6. **PoE + DC Redundant Power** - Dual power input (PoE version)
7. **Local Indonesia Support** - Technical support dalam Bahasa Indonesia
