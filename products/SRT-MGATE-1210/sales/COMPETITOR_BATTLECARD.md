# BATTLECARD BY COMPETITOR

## vs USR-IOT N720-ETH

### Ketika Customer Menyebut USR-IOT

**Customer:** "USR-N720 lebih established brand"

**Response:**
> "USR-N720 memang brand established dari China. Tapi untuk fitur yang sama, SURIOTA punya keunggulan:
>
> | Aspect | USR-N720 | SURIOTA |
> |--------|----------|---------|
> | Harga | Rp 3.5M | **Rp 2.7M (23% cheaper)** |
> | WiFi | ❌ Tidak ada | **✅ Built-in** |
> | BLE Config | ❌ Tidak ada | **✅ Setup via HP** |
> | Support | China (bahasa Inggris) | **Indonesia (Bahasa Indonesia)** |
>
> Dengan harga lebih murah, Anda dapat WiFi backup dan konfigurasi via smartphone yang USR tidak punya."

### USR-IOT Weaknesses

| Weakness | SURIOTA Advantage |
|----------|-------------------|
| No WiFi | Built-in WiFi + auto failover |
| Web config only | BLE mobile config |
| No MQTT native | MQTT native support |
| China support | Local Indonesia support |
| Legacy protocol focus | Modern IoT-ready |

### Kapan USR-IOT Lebih Baik

- Customer sudah invest ecosystem USR
- Need serial-to-Ethernet saja (bukan IoT)
- Super tight budget (USR-N540 @ Rp 1.8M)

**Pivot:** "Untuk IoT/Cloud use case, SURIOTA lebih cocok dengan WiFi dan MQTT native."

# WIN THEMES BY SCENARIO

## Scenario: Retrofit SCADA ke Cloud

**Lead with:**
- MQTT native output
- Multi-cloud support (AWS, Azure, Datacake, dll)
- Tidak perlu ubah infrastruktur existing

**Key differentiator:** "Bridging gap antara legacy Modbus dan modern cloud"

---

## Scenario: Remote Monitoring di Site Terpencil

**Lead with:**
- WiFi + Ethernet dual connection
- Auto failover
- BLE config (tidak perlu laptop ke site)

**Key differentiator:** "Setup dan troubleshoot dari HP, tidak perlu bawa laptop"

---

## Scenario: Multi-Site Deployment

**Lead with:**
- Harga kompetitif untuk volume
- Standardized mobile app config
- Central cloud management

**Key differentiator:** "Same app, same config flow untuk semua site"

---

## Scenario: System Integrator

**Lead with:**
- Margin opportunity (distributor program)
- Flexible cloud (customer choice)
- Local technical support

**Key differentiator:** "Partner dengan produk lokal, margin lebih baik"
